You need to install the Lokiaux extension in order to use the clipboard. Here's how:

  1. Download the extension.
  2. If a yellowish bar appears that says, "To protect your computer, Firefox prevented this site from installing software on your computer,"
    • Click "Edit Options..."
    • Make sure apps.carleton.edu is listed in "Address of web site", then click "Allow", then "Close".
    • Then download the extension again.
  3. In the dialog that appears, press "Install".
  4. Close all browser windows and restart the browser.

The Lokiaux extension will allow pages at carleton.edu to access your clipboard; other sites will still be denied access.
